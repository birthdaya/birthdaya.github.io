最小生成树Kruskal算法

并查集

定义

并查集是一种树形数据结构，支持两种操作：

合并：将两个子集合并成一个子集
查找：确定某个元素处在哪个集合
fa[x]fa[x]fa[x] 存储节点 xxx 的父节点.初始化时，每个节点是一个集合，每个节点的父节点是他自己

两种操作

查找

根节点是集合的代表，查找就是找到元素所在集合的根

int find(int x){  
	if(fa[x]==x) return x;  
	return find(fa[x);  
}

缺点：耗时，在查找时链条较长。
优化：带路径压缩的查找，在返回的路上顺带修改父节点的根

int find(int x){  
	(fa[x]==x) return x;  
	return fa[x]=find(fa[x]);  
}

合并

合并就是把一个集合根指向另一个集合的根，只需要修改根即可

void union1(int x,int y){  
	fa[find(x)]=find(y);  
}

Kruskal算法

Kruskal 算法是利用并查集求MST（最小生成树）的算法

数据定义

e[i]e[i]e[i] 存储第 iii 条边起点、终点、边权。fa[x]fa[x]fa[x] 存储节点 xxx 的父节点

算法步骤

1 .初始化并查集，每一个节点是一个集合，每一个节点的父节点是他自己
2. 将所有边按边权大小从小到大排序
3. 按顺序枚举每一条边，如果这条边连接的两点不在同一集合，就把这条边加到最小生成树，并且合并两个集合。如果这条边连接的两点在同一集合，则这条边是废边，跳过
4. 重复执行步骤 333 直到选取 n−1n-1n−1 条边为止（最小生成树是 nnn 个节点 n−1n-1n−1 条边，如果是nnn 条边就是一个环）

代码

#include<bits/stdc++.h>  
using namespace std;  
const int N=2*1e5+10;  
int n,m,fa[N],ans,cnt;  
struct edge{  
	int u,v,w;  
	bool operator<(const edge &a)const{  
		return w<a.w;  
	}  
}e[N];  
int find(int x){  
	if(fa[x]==x) return x;  
	return fa[x]=find(fa[x]);  
}  
bool kruskal(){  
	sort(e,e+m);  
	for(int i=1;i<=n;i++) fa[i]=i;  
	for(int i=0;i<=m;i++){  
		int x=find(e[i].u);  
		int y=find(e[i].v);  
		if(x!=y){  
			fa[x]=y;  
			ans+=e[i].w;  
			cnt++;  
		}  
	}  
	return cnt==n-1;  
}  
int main(){  
	cin>>n>>m;  
	for(int i=0;i<m;i++){  
		cin>>e[i].u>>e[i].v>>e[i].w;  
	}  
	if(kruskal()) cout<<ans;  
	else cout<<"orz";  
	return 0;  
}
